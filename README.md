# lint-kit Monorepo

This is the monorepo for [lint-kit](./packages/core), a comprehensive linting suite.

## Structure

- **packages/core**: The core library.
- **examples/**: Example projects demonstrating usage with various frameworks.
- **docs/**: Documentation guides.

## Development

```bash
# Install dependencies
pnpm install

# Build all packages
pnpm build

# Lint all packages
pnpm lint
```
